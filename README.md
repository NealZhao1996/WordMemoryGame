# WordMemoryGame
A Word Memory Game built with Next.js &amp; React

## Getting Started

To run the development server:

```bash
npm install
# then
npm run dev
```

To run with Docker:
```bash
docker build -t word-memory .
# then
docker run --publish 3000:3000 word-memory
```
